## ۱. واژگان پایه — Vocabulary

| نماد | تعریف فارسی | Definition |
|------|-------------|------------|
| `H(x)` | x انسان است | x is a human |
| `G(x)` | x خداست | x is God |
| `Act(a, α)` | a فعل α را انجام می‌دهد | a performs action α |
| `Ans(a, α, t)` | a در t پاسخگوی α است | a is accountable for α at t |
| `Final(t)` | t روز داوری نهایی است | t is the Final Judgment |
| `Hidden(t)` | زمان t پنهان است | the time t is unknown |
| `UltAuth(a, b)` | a سلطه مطلق بر b دارد | a has ultimate authority over b |
| `Resp(a, α)` | a شخصاً مسئول α است | a is personally responsible for α |
| `Seal(p)` | p آخرین پیامبر است | p is the final prophet |
| `NewProphet(q)` | q ادعای نبوت جدید دارد | q claims new prophethood |
| `CompulsoryGuide(a,b)` | a به‌اجبار b را هدایت می‌کند | a compulsorily guides b |
| `Gap(Is, Ought, t)` | شکاف بین هست و باید در t | gap between Is and Ought at t |
| `Guide(g, t)` | g هادی در زمان t است | g is a guide at time t |
| `State(s)` | s دولت/حاکمیت انحصاری است | s is a monopoly state |
| `Error(s, α)` | s در α خطا کرده است | s has made error α |
| `PaysPrice(c, α)` | c هزینه α را می‌پردازد | c pays the cost of α |
| `𝓕` | نظام صوری آزادی | Formal System of Freedom |
| `Ax` | مجموعه اصول موضوعه | Set of axioms |
| `⊢` | رابطه استنتاج | derivability relation |
| `∃!` | وجود یکتا | unique existence |

## ۲. آکسیوم‌های دینی — Religious Axioms

### A1 — توحید / Tawhid

**صورت الهیاتی:**
```
∃! x  G(x)
```

**خوانش سیاسی-حقوقی:**
```
∀a ∀b (H(a) ∧ H(b)) → ¬UltAuth(a, b)
```

**معنا:**
خدا یگانه است؛ هیچ انسانی خدا نیست. پس هیچ انسانی حق ندارد منشأ سلطه مطلق بر انسان دیگر باشد. توحید = ضمانت ساختاری آزادی. دولت مطلق‌العنان، ولایت مطلقه فقیه، و هر شکل دیگری از سلطه انسان بر انسان مستقیماً این آکسیوم را نقض می‌کنند.

---

### A2 — معاد / Ma'ad

**وجود روز داوری نهایی:**
```
∃t (Final(t) ∧ Hidden(t))
```

**پاسخگویی جهانی افعال:**
```
∀a ∀α (Act(a, α) → Ans(a, α, t))
```

**معنا:**
داوری نهایی وجود دارد اما زمانش پنهان است. هر فعل پاسخ دارد. «مجهول بودن پایان بازی» محاسبه جرم را ناممکن می‌کند و نظم آزاد را از مشکل *End Game* در نظریه بازی‌ها نجات می‌دهد.

---

### A3 — نبوت / Nubuwwat

**یکتایی نظام صوری آزادی:**
```
∃! ConsistentFormalSystem(Freedom)
```

**ختم نبوت — بستن باب رسالت‌سازی:**
```
∃p (H(p) ∧ Seal(p))
  → ∀q (H(q) ∧ q ≠ p → ¬NewProphet(q))
```

**نفی هدایت اجباری:**
```
∀a ∀b (H(a) ∧ H(b) → ¬CompulsoryGuide(a, b))
```

**معنا:**
نظام صوری منسجم برای آزادی یکتاست و تکرارناپذیر. وحی نبوی ختم شده؛ هر مدعی رسالت جدید یا نظام صوری ناسازگار می‌آورد یا کلاهبردار است. *لا اکراه فی‌الدین*.

---

### A4 — عدل الهی / Divine Justice

**اصل شخصی‌بودن تاوان:**
```
∀a ∀α ∀b (Act(a, α) ∧ a ≠ b → ¬Resp(b, α))
```

**ناسازگاری ساختاری دولت با عدل:**
```
State(s) ∧ Error(s, α)
  → ∃c (¬Responsible(c, α) ∧ PaysPrice(c, α))
```

**معنا:**
هر فرد تنها مسئول عمل خویش است. دولت این اصل را ساختاری نقض می‌کند: خطا از بالا رخ می‌دهد، هزینه در پایین توزیع می‌شود (از طریق تورم، مالیات، جنگ، فقر).

---

### A5 — امامت / Imamat

**ضرورت هدایت در شکاف هست → باید:**
```
∀t (FreedomSystem(𝓕) ∧ Gap(Is, Ought, t)
  → ∃g Guide(g, t))
```

**تضمین دوطرفه ورود/خروج از بردگی:**
```
∀a (H(a) →
  (WantsFree(a) → CanBeFree(a))
  ∧
  (InSlavery(a) → CanReturnFree(a))
)
```

**معنا:**
نظام صوری بدون هادی برای پر کردن شکاف «خواست» و «واقعیت» ناقص است. امامت پرکننده این شکاف است. جاده دوطرفه ورود/خروج از بردگی همیشه باید باز باشد.

## ۳. فرا-اصول نظام صوری — Meta-Axioms

### M1 — عینیت و بیان‌پذیری صوری

```
∃𝓕 (FormalSystem(𝓕) ∧ Objective(𝓕) ∧ Expressible(𝓕))
```

**معنا:**
دین/آزادی باید در قالب نظام صوری عینی و بیان‌پذیر صورت‌بندی شود، نه تفسیرهای ذوقی و بی‌پایان. «عینی» = قابل توافق بین‌الاذهانی. «بیان‌پذیر» = قابل ترجمه به هر زبان بشری یا ماشینی.

---

### M2 — سازگاری / Consistency

```
Consistent(𝓕) :=
  ¬∃φ (𝓕 ⊢ φ ∧ 𝓕 ⊢ ¬φ)
```

**معنا:**
از نظام نباید هم یک گزاره و هم نقیضش استنتاج شود. هیچ نتیجه‌ای نباید اصل آزادی را نقض کند. اکثر نظریه‌های سیاسی موجود این شرط را برآورده نمی‌کنند.

---

### M3 — متناهی و حداقلی / Finite & Minimal

**شرط اقتصاد منطقی:**
```
Finite(Ax) ∧ Minimal(Ax)
```

**اصل اُکام — ترجیح نظام کم‌اصل‌تر:**
```
∀ 𝓕₁, 𝓕₂ (
  SameGoal(𝓕₁, 𝓕₂)
  ∧ |Ax(𝓕₁)| < |Ax(𝓕₂)|
  → RationallyPreferred(𝓕₁)
)
```

**معنا:**
اگر دو نظام یک هدف را دنبال کنند، نظامی که اصول کمتری دارد از نظر عقلانی برتر است. هر اصل اضافی پیچیدگی تصاعدی ایجاد می‌کند و نیاز به مفسر می‌آورد.

## ۴. فرمول‌های زنجیره‌ای — Causal Chains

### C1 — نظم اخلاقی ↔ پول سالم

**مسیر مثبت:**
```
نظم اخلاقی
  → کاهش رجحان زمانی
  → مهار دوپامین‌محوری
  → مهار تورم
  → رقابت پولی
  → پول خصوصی
```

**مسیر منفی:**
```
فساد اخلاقی
  → دوپامین‌محوری جامعه
  → رجحان زمانی بالا + استرس
  → تورم + ربا + پول انحصاری
  → مداخله دولت
```

> **نکته:** ربا = فشرده‌ترین صورت این زنجیره. پول فاسد و انحطاط اخلاقی در حلقه بازخورد خود-تقویت‌گر قرار دارند.

---

### C2 — دولت ↔ آزادی: دو زنجیره متضاد

**زنجیره دولت (بردگی):**
```
دولت
  → پیش‌بینی‌پذیرسازی
  → عادی‌سازی جبر
  → تضعیف اراده و مسئولیت
```

**زنجیره آزادی (خلاقیت):**
```
آزادی
  → پیش‌بینی‌ناپذیری
  → تقویت اختیار
  → افزایش مسئولیت و خلاقیت
```

> **نکته:** این ادعا روان‌شناختی است، نه صرفاً حقوقی. دولت از طریق عادت‌سازی به جبر، ظرفیت اراده آزاد را در سطح رفتاری تضعیف می‌کند.

## ۵. قضایای مشتق‌شده — Derived Theorems

قضایای زیر از ترکیب آکسیوم‌های بالا قابل استنتاج‌اند:

**T1 — نفی دولت توتالیتر** (از A1):
```
∀s (State(s) ∧ UltAuth(s, ∀a H(a)) → Violates(s, A1))
```

**T2 — پایداری نظم آزاد** (از A2):
```
Hidden(t_final) → ¬Calculable(Crime(a)) → StableOrder(Freedom)
```

**T3 — ممنوعیت رسالت‌سازی جدید** (از A3 + M2):
```
NewClaim(q, FormalSystem') ∧ FormalSystem' ≠ 𝓕
  → ¬Consistent(FormalSystem')
```

**T4 — مسئولیت فردی اقتصادی** (از A4):
```
Tax(s, a) ∧ ¬Consent(a) → Violates(s, A4)
```

**T5 — حق خروج از هر نظام** (از A5):
```
∀a ∀System (InSystem(a, System) → CanExit(a, System))
```

## ۶. شرایط سازگاری — Consistency Conditions

یک نظام سیاسی/حقوقی **منسجم** است اگر و تنها اگر:

```
Consistent(System) ⟺
  ∀ Theorem T ∈ System:
    (1) T is derivable from {A1, A2, A3, A4, A5}
    ∧ (2) T does not contradict any Aᵢ
    ∧ (3) Finite(Ax(System))
    ∧ (4) Objective(System)
```

**نقض‌کننده‌های شناخته‌شده:**

| نظام | آکسیوم نقض‌شده | دلیل |
|------|----------------|------|
| دموکراسی مطلق | A1, A4 | اکثریت می‌تواند حق اقلیت را نقض کند |
| کمونیسم | A1, A4 | سلطه مطلق حزب، توزیع تاوان اشتباه |
| ولایت مطلقه فقیه | A1, A3 | سلطه مطلق انسان، ادعای رسالت جدید |
| لیبرالیسم بی‌دین | A2, M3 | حذف A2 نیاز به اصول اثبات‌ناپذیر بیشتر دارد |
| عرفان سیاسی | M1, M2 | نظام غیرعینی و ناسازگار درونی |
